Список дел и сортировка 
	{
		Класс CardProvider

		Список дел - Dictionary pool <id карты, карта>

		Хранятся и сортируются только индексы карт для экономии памяти

		Все отсортированы по убыванию даты обновления:

		idListDefault - только невыполненные без приоритета

		idListPriority - только невыполненные с приоритетом //Остортированы по приоритету

		idListCompleted - только выполненные

		Конструктор из заданного списка

		Методы получения разных списков карт по id

		Поиск по части названия или описания

		Сохранение, удаление карты

		Сортировка всех списков
		1. Словарь сортируется по дате обновления
		2. Все типы карт(выполненные, с приоритетом, остальные) помещаются в свои списки
		3. Карты с приоритетом сортируются по приоритету
